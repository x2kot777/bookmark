# @En_learner0526 (勉三さん@🇺🇸MBA&英語学習) - X 投稿

Author: 勉三さん@🇺🇸MBA&英語学習 (@En_learner0526)

Tweet text:
育休中に毎日GPT英会話練習してたけど、結構スピーキング上手くなったなと今日の国際会議で自画自賛（GPTにCEFR判定してもらったら、B2からC1にも上がってたし）

以下、私のルーティーン（トータルで50-60分くらいです）

①5-10分程度の英会話ニュースを字幕なしで見る
②動画のスクリプトを音読して、わからなかった単語をピックアップ
③GPTにスクリプトをアップロードして理解度確認の問題を三問出してもらいスピーキングで回答
④動画の内容を一分程度で要約
⑤GPTにストラクチャーを可能な範囲で維持してもらい、添削バージョンを作ってもらう
⑥添削バージョンをほぼ暗唱できるまで音読
⑦動画の内容についてGPTに問題を作ってもらい、フリーディスカッション（間違いや不自然なワードは指摘するように指示。英語でうまく言えなかった表現についてもここで質問）
⑧②でピックアップした単語の解説と例文を日英で作成してもらい、例文をAnkiに登録
⑨Ankiの例文で瞬間英作文を実施（大体15-20問くらい）

特に育休中に始めた⑦、⑧、⑨のお陰で自分がうまく言えない表現が浮き彫りになってきてスピーキングの幅が広がったなぁと　

たとえばビジネスだとleverage とかraise concerns とかってめっちゃ使い勝手良くて例文使うとすぐに身につくんですよね

あと同じ動画でのシャドーイングも日々の学習に入れたりしましたが、これはどんだけ効果あったかよくわからんなぁ…

TOEFLの勉強してたときにイギリス人の講師から「全部の単語聞き取れなくても大丈夫。全体像を見失わないように意識して聞け！」と指導されましたが、やっぱニュース（特に素人のインタビューとか）だと何回聞いても聞き取れない部分あるんですよね…
まぁ多少聞き取れなくても致命傷にはならんかなぁと実世界でも思ってます
