# @ohno_katsuya (大野克也｜ソロプレナーという生き方) - X 投稿

Author: 大野克也｜ソロプレナーという生き方 (@ohno_katsuya)

Tweet text:
1人で750万円/月を稼ぐスモビジオーナー、アメリカのPieter Levelsさんは何がすごいのか。

すごいのは、「事業を量産して、当たりだけ残す」を徹底してることです。

▼何がすごいのか
・Nomad List（リモートワーカー向けの都市比較サービス）
・Remote OK（リモート求人サイト）
・ほかにも20以上のサービスを一人で運営

彼はエンジニアの専業出身ではありません。
独学でコードを書きながら、企画・開発・運用・集客まで全部を一人で回しています。

社員を雇わないので、売上がほぼそのまま利益として残る。
年商は約300万ドル（約4.6億円）に達しますが、本当に異常なのは金額より「これをほぼ一人で生み出している」という構造です。

普通なら数十人の会社がやる仕事量を、彼はたった一人でさばいている。
口癖は「完璧を待つな、雑に出して反応を見ろ」です。

▼なぜ一人で勝てるのか
・打席数で勝つ
・伸びる市場の“周辺”を取る
・在庫ゼロで失敗が数千円

まず「打席数」。
1つの大ヒットを狙うのではなく、たくさん出して数個当たればいい、という発想です。
1勝9敗でも、その1勝が大きければトータルで勝てる。

次に「周辺を取る」。
「働く場所の自由化」という巨大トレンドのど真ん中ではなく、その周りの“困りごと”（どの街が住みやすい？どこに求人がある？）を取りにいきました。

最後に「在庫ゼロ」。
デジタルなので、外しても損失は数千円のサーバー代程度。
だから何度でも打席に立てます。

▼他にもいる「一人で数億」の個人
・Justin Welsh…年商約500万ドル（約7.7億円）
・Dan Koe…利益率98%、年商約260万ドル（約4億円）
・Kat Norton（Miss Excel）…年商約200万ドル（約3億円）

3人に共通するのは、「自分の経験やスキルを、在庫のない商品にして、一人で回している」こと。

発信や講座は、一度作れば何度でも売れます。
仕入れも在庫もなく、利益率が極端に高い。
これが、社員を雇わなくても個人が数億規模に届く理由です。

▼日本でやるなら
・伸びる領域の“困りごと検索”を比較メディアで取る
・撤退ラインを数千円に決めて3〜5個を並行テスト
・当たった1個に広告と外注を一点集中

たとえば「副業」「介護」「リスキリング」のような伸びている領域では、ユーザーが必ず何かに困って検索しています。
その“困りごと”を、比較メディアやまとめで解決する。

最初から1本に賭けず、撤退ラインを数千円に決めて3〜5個を同時に試す。
反応が出た1個にだけ、広告と外注を集中させる。
これがPieter流「量産して当たりに寄せる」の日本版です。

要は金脈を掘るな、金脈の横でスコップを売れ。
僕がファクタリング比較メディアで月商1,000万を超えたのも、まったく同じ構造でした。
